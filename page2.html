Week 1: building a Website

This week's Project was all about building a website! I have zero experience with HTML and coding so this week was extremely challenging for me, but eventually I started making my way through it. Heres my progress! First, I started out with downloading the tools I needed to start my journey. The tools I needed were GitBash, Visual Studio Code, a GitHub account and a who lot of willpower. I started by installing the programs and fiddling around with HTML for the first time. Then, I started inserting the template I downloaded off of the W3CSS website, and started to customize it. I would tinker with the code at first, but I had no prior coding experience with HTML, so this was all new to me. I started making changes slowly, getting a feel for how it all worked. I felt lost, with nowhere to turn. I ended up having to call some of my computer science friends to help me out, and i'm glad I did because I felt like a lost puppy. So after these problems, I started to really try to get my ontop of the code. After I started changing things, I got the hang of it and ended up even adding images on my own. Soon after, I kept on adding, and adding, that I needed to make new pages to sort through everything I had added. I hate using cluttered web pages, so I wanted to keep it as clean as possible. I soon added multiple pages, including a home page where it's just a gif of a 3d printer and a menu that includes a projects page, a link to the home page, and an about me page. I continued to tinker with my code to get all my formatting as close to perfect as I could, centering text and images. After getting the centering down on most pages, I started putting the final touches on all of my code for the base website. Finally, I pushed everything out to the actual Github site and, Voila! Website done. Althogh, at the time of writing this, the images did not display due to GitHub's delay in file sync.
